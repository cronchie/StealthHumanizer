# FAQ

## Does StealthHumanizer store my text or API keys on a server?

The project is designed to keep provider keys in client-side browser storage and avoid intentional server-side persistence of user content.

## Why does integration smoke fail on a fresh clone?

`npm run test:integration` expects benchmark and model manifests under `data/*/current/` generated by benchmark/training scripts.

## Why can `npm run build` fail in restricted environments?

If the runtime cannot access external font/CDN endpoints, build-time asset fetching can fail.

## Which branch should pull requests target?

All pull requests should target `main`.

## Where should I report vulnerabilities?

Use the private disclosure flow in [SECURITY.md](https://github.com/rudra496/StealthHumanizer/blob/main/SECURITY.md).
