# FAQ

## Does StealthHumanizer store my text or API keys on a server?

No. Provider API keys are stored in your browser's localStorage only. Text goes directly from your browser to the AI provider you select. No user content is persisted server-side.

## Which providers are free?

Gemini, Groq, Mistral (limited), Cohere (limited), Together, OpenRouter, Cerebras, DeepInfra, ZAI, HuggingFace, and Cloudflare all offer free tiers. The CLI runners (`claude-code`, `codex`) are also free if you have those tools installed locally.

## Can I use StealthHumanizer without an API key?

Yes, for detection only. The `detect` command in the CLI and the Detector tab in the web UI use the built-in heuristic engine — no API key needed. For humanization, you need at least one provider API key (or a local CLI tool like Claude Code).

## How do I get a free API key?

1. **Gemini** — Go to [Google AI Studio](https://aistudio.google.com/apikey) and create a key.
2. **Groq** — Go to [Groq Console](https://console.groq.com/) and create a key.

Both offer generous free tiers and work immediately.

## Why does my humanized text still get flagged by detectors?

No humanization tool can guarantee 100% bypass. Results depend on:
- The rewrite level you chose (try `aggressive` or `ninja`)
- The AI provider and model
- The input text complexity
- The specific detector being used

Use the built-in detection scoring to iterate and improve results.

## What is the difference between rewrite levels?

- **Light** — Subtle edits, minimal changes to your text
- **Medium** — Natural rewrite, good balance of change and preservation
- **Aggressive** — Complete rewrite with strong fingerprint disruption
- **Ninja** — Maximum stealth with multi-pass auto-refinement targeting 90%+ human score

## What are protected regions?

When enabled, code blocks, URLs, links, and @mentions in your text are preserved verbatim during humanization. This prevents the AI from altering technical content, citations, or references.

## How do I use the CLI?

```bash
# Install and build
npm run cli:build

# Detect AI content (no API key needed)
stealthhumanizer detect --text "Your text here"

# Humanize with a provider
stealthhumanizer humanize --model gemini --level aggressive -i draft.txt -o output.txt

# Use Claude Code locally (no API key)
stealthhumanizer humanize --model claude-code -i draft.txt
```

## Can I self-host StealthHumanizer?

Yes. Clone the repo, run `npm install`, set your environment variables (see `.env.example`), and run `npm run dev` or deploy to Vercel with `vercel deploy`.

## Why does integration smoke fail on a fresh clone?

`npm run test:integration` expects benchmark and model manifests under `data/*/current/` generated by benchmark/training scripts. Run the pipeline scripts first or skip this test.
