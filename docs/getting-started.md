# Getting Started

## Prerequisites

- Node.js 20+
- npm 10+

## Installation

```bash
git clone https://github.com/rudra496/StealthHumanizer.git
cd StealthHumanizer
npm ci
```

## Run Locally

```bash
npm run dev
```

Open `http://localhost:3000`.

## First Use

1. Add provider API keys in Settings.
2. Paste source text.
3. Choose rewrite level/style/tone.
4. Run humanization and review detector output.
